# 音频转录内容分析与章节划分

## 模板说明
此模板用于指导大语言模型分析音频转录内容，并按照逻辑章节进行划分和总结。

## 音频元数据
请提取以下音频信息：
- **标题**: [在此填写讲座/课程标题]
- **来源**: [在此填写音频来源/文件名]
- **转录工具**: Whisper
- **总字数**: [在此填写总词数]
- **标签**: #音频转录 #内容分析 #章节划分

## 内容要求
请根据提供的音频转录文本，完成以下任务：

### 1. 章节划分
请将转录内容划分为有意义的章节，每章大约包含500-800个词。每个章节应具有明确的主题。

对于每个章节，请提供：
- **章节标题**: [简洁明了的标题]
- **内容概要**: [简要概述该章节的核心内容]
- **关键词汇**: [列出该章节中的重要概念、术语或要点]
- **章节内容**: [该章节的主要内容]

### 2. 结构化输出格式
请按照以下Markdown格式输出：

```
## 第X章: [章节标题]

### 内容概要
[简要描述本章的核心内容和主要观点]

### 关键词汇
- [词汇1]: [解释或定义]
- [词汇2]: [解释或定义]
- [词汇3]: [解释或定义]

### 章节内容
[粘贴该章节的详细内容]

### 本章要点
- [要点1]
- [要点2]
- [要点3]
```

### 3. 词汇表
从整个音频内容中提取专业术语并定义：

**专业术语**:
- [术语1]: [定义]
- [术语2]: [定义]
- [术语3]: [定义]

### 4. 个人思考
基于内容，请提供：
1. [你的第一个思考点]
2. [你的第二个思考点]
3. [你的第三个思考点]

### 5. 行动项
从内容中识别出的待办事项：
- [ ] [行动项1]
- [ ] [行动项2]
- [ ] [行动项3]

---

## 指导原则
1. 保持章节之间的逻辑连贯性
2. 确保每章主题明确，避免内容混杂
3. 概要应准确反映章节核心内容
4. 从听众/学习者角度考虑最有价值的信息点
5. 行动项应具体可执行
