# 舆情分析技能 - 更新日志

## 2026-03-24 - 版本 1.2.0

### 优化内容

#### ✅ 去掉"30 条"强制要求
- **变更**：不再强制要求输出 30 条明细数据
- **新策略**：实事求是，百炼 API 返回多少条就展示多少条
- **原因**：某些冷门主题可能公开数据较少，强制 30 条会导致数据质量下降

#### ✅ 文档更新
- `SKILL.md`：移除"30 条"相关描述，改为"明细数据表（数量由百炼 API 返回决定）"
- `README.md`：更新输出格式说明，明确数据条数由 API 决定
- `run.sh`：脚本自动统计并显示实际数据条数

#### ✅ 配置保持不变
- App ID: `cd6799ccd4fe45fd984ff00f51653f7a`
- 超时时间：300 秒（5 分钟）
- 工作模式：透传模式
- 重试机制：2 次

### 文件变更

| 文件 | 变更内容 |
|------|----------|
| `SKILL.md` | 移除"30 条"强制要求，改为"数量由百炼 API 返回决定" |

### 输出格式变更

#### 变更前
```
【明细数据】
- 30 条明细数据表（强制输出 30 条）
```

#### 变更后
```
【明细数据】
- 数据条数：X 条（由百炼 API 根据实际可获取信息决定）
```
